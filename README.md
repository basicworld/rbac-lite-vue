# rbac-lite-vue

基于 vue 的 rbac 权限系统

## 开发环境启动方式

依赖 npm 版本=6.9.0

1. 使用 VSCode 加载工程
2. 执行`npm run dev`
3. 成功启动后在浏览器访问`http://localhost:9528/`，输入用户名 admin、任意密码、验证码，即可登录
